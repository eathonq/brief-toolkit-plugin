## 快速开始

### [I18n面板]
I18n面板提供了一个界面来管理和编辑项目中的国际化资源。
您可以在这个面板中添加\转移多语言资源目录，新增、设置模板、同步、打开和删除国际化资源文件。

- 多语言资源目录添加：会在 resources 目录下创建一个新的目录，和创建一份 .schema.json 多语言模板；
- 多语言资源目录转移：会将原来目录下的文件移动到新的目录下；

- 多语言资源文件新增：会在当前目录下创建一个新的国际化资源文件，并设置 meta 属性；
- 多语言资源文件设置模板：会将当前目录下的国际化资源文件设置为指定模板的格式；
- 多语言资源文件同步：会将当前目录下的国际化资源文件作为标准，和同目录下的其他国际化资源文件进行同步，
添加新增的属性，删除多余的属性；
- 多语言资源文件打开：会在编辑器中打开当前目录下的国际化资源文件；
- 多语言资源文件删除：会删除当前目录下的国际化资源文件；

### [Guide面板]
Guide面板提供了一个界面来管理和编辑项目中的引导资源。
您可以在这个面板中添加\转移引导资源目录，新增、打开和删除引导资源文件。

- 引导资源目录添加：会在 resources 目录下创建一个新的目录，和创建一份 .schema.json 引导模板；
- 引导资源目录转移：会将原来目录下的文件移动到新的目录下；
- 引导资源文件新增：会在当前目录下创建一个新的引导资源文件，并设置 key 属性；

## BriefToolkit 组件组
BriefToolkit 组件组提供了一些常用的组件来帮助开发者更方便地开发和管理国际化资源和引导资源。

### Guide 组件分组：
提供了一个界面来展示引导资源的内容，支持多语言显示和引导资源的切换。
- **GuideManager 组件**
  该组件提供了引导任务的管理功能，包括任务的启动、加载和记录查询。

### I18n 组件分组：
提供了一个界面来展示国际化资源的内容，支持多语言显示和国际化资源的切换。
- **LocalizedLabel 组件**
  该组件实现了本地化文本的功能，支持Label、RichText和EditBox组件。
- **LocalizedManager 组件**
  该组件实现了多语言切换的功能，支持在编辑器模式下修改默认语言资源和资源路径。
- **LocalizedSprite 组件**
  该组件实现了本地化图片的功能，支持Sprite组件。
  
### Mvvm 组件分组：
提供了一些常用的组件来帮助开发者更方便地开发和管理国际化资源和引导资源。
- **Binding 组件**
  该组件用于绑定组件元素值与数据属性的关联，并可以选择绑定模式。支持双向绑定、单向绑定和一次性绑定等多种模式，适用于不同的使用场景。
- **DataContext 组件**
  该组件用于绑定上级数据中的对象数据到组件上，提供数据上下文管理功能。
- **ItemsSource 组件**
  该组件用于绑定上级数据中的集合数据到组件上，提供数据集合管理功能。
- **ViewModel 组件**
  该组件用于管理视图模型的数据绑定和生命周期。

### UIM 组件分组：
提供了一些常用的组件来帮助开发者更方便地开发和管理国际化资源和引导资源。 
- **AudioManager 组件**
  该组件实现了音频管理的功能，包括背景音乐和音效的播放、暂停、停止以及音量控制等功能。
- **MessageBoxBase 组件**
  该组件提供了消息框的基础功能，包括显示标题、内容和不同类型的按钮，并处理用户的交互结果。
- **SkinManager 组件**
  该组件实现了皮肤管理的功能，包括主题的切换、皮肤项的激活和状态管理等。
- **SkinSprite 组件**
  该组件用于在 Cocos Creator 中显示皮肤相关的精灵，自动根据指定的皮肤标识加载对应的 SpriteFrame。
- **TooltipBase 组件**
  该组件提供了提示框的基础功能，包括显示内容文本、关闭按钮，并处理用户的交互结果。
- **TooltipMultiple 组件**
  该组件提供了多提示框的基础功能，包括显示多个提示框实例、管理提示框的显示顺序和关闭操作。
- **ViewBase 组件**
  该组件提供了视图的基础功能，包括视图类型、缓存选项和基本的关闭/返回事件处理，为其他具体视图组件提供统一的基础。
- **ViewManager 组件**
  该组件实现了 IViewManager 接口，提供了视图管理的核心功能，包括视图的显示、隐藏、关闭、数据通知等操作，并支持视图的层级管理（View、MessageBox、Tooltip）。通过该组件，开发者可以方便地在项目中管理各种类型的视图，并且可以在编辑器中配置默认视图和预制体列表。
- **ViewSort 组件**
  该组件提供了视图排序的绑定功能，通过设置视图的排序索引来控制视图在不同层级的显示顺序，确保用户界面元素按照预期的层级关系进行渲染。
